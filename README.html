A Claude skill that writes ATS-optimized resumes for quant finance and tech roles — with a built-in review loop, JD-mirror mode, and zero fabrication.

What it does

🎯

JD-Mirror Mode

Ingests a job posting, extracts required keywords, and rewires every bullet to match ATS scoring verbatim — without fabricating.

🔍

3-Pass Review Loop

ATS gate → human voice check → hiring manager scan. Runs internally before any output so the resume reads like a person wrote it.

🚫

Zero Fabrication

Hard rule — refuses to invent skills, titles, years of experience, or metrics. Every fact is traceable to what you provided.

📄

DOCX Output

ATS-safe Word file on request. No tables, no columns, correct bullet format — won't fail Workday or Greenhouse parsers.

Architecture

How it works

1

Profile intake

Claude asks for everything in one message: education, experience, projects, skills, target role. Never invents what you don't provide.

2

Mode selection

Auto-detects mode from your input. Paste a JD → JD-Mirror activates. Paste your resume → Optimize. Ask for a score → Score mode.

3

JD-Mirror keyword extraction (if JD provided)

Extracts required vs preferred keywords from the job posting. Builds a match report. Injects keywords verbatim into bullets — same spelling, same capitalization as the JD.

4

Draft generation

Writes resume following strict structure: Header → Education → Experience → Projects → Skills. All bullets follow Action Verb → Task → Quantified Result.

5

3-Pass internal review

ATS Gate (keywords, dates, formatting) → Human Voice Check (no AI patterns, varied rhythm, specific details) → Hiring Manager Scan (narrative, top-half impact). Revises and re-runs if anything fails.

6

Output

Plain text by default (copy-paste into any ATS portal). Add "give me a .docx" for an ATS-safe Word file with correct formatting.

🔄 The 3-Pass Review Loop — what makes this different

Most AI resume tools output on the first pass. This skill runs three internal checks before producing any output:

  • Pass 1 — ATS Gate: keywords, date format, section headers, bullet structure, zero fabrication
  • Pass 2 — Human Voice Check: no passive voice, no identical rhythm, no AI filler words, specific proper nouns
  • Pass 3 — Hiring Manager Scan: strongest line on page 1, logical narrative arc, skills match bullets

If any check fails → revise → re-run all 3 passes. Only then does output happen.
