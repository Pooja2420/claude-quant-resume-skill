# Banned Phrases & Clichés

Remove ALL of the following from any resume. They harm ATS scores and signal weak writing.

---

## Personality Fluff (Delete entirely)

- team player
- go-getter
- self-starter
- detail-oriented
- hardworking / hard-working
- dedicated
- passionate about
- enthusiastic
- results-driven / results-oriented
- dynamic
- proactive
- highly motivated
- ambitious
- driven
- people person
- strong work ethic
- fast learner / quick learner
- eager to learn
- thrive in fast-paced environments
- outside the box
- think outside the box
- synergy
- leverage (as a verb)
- value-add
- move the needle
- best of breed
- mission-critical
- robust (when describing yourself)
- thought leader

---

## Weak Ownership Phrases (Rewrite with active ownership)

| ❌ Banned | ✅ Replacement pattern |
|-----------|----------------------|
| Responsible for X | [Action verb] X |
| Duties included X | [Action verb] X |
| Helped with X | [Specific contribution] to X |
| Assisted in X | [What you specifically did] |
| Worked on a team to X | Led / Developed / Contributed [specific part] |
| Involved in X | [What you did in X] |
| Part of a team | remove or specify: "as one of 3 engineers, built..." |
| Participated in | remove or rewrite with action |
| Supported X | [What you specifically did to support] |
| Contributed to X | [Specific contribution] |

---

## Vague Communication Claims (Delete or replace with evidence)

- excellent communication skills
- strong interpersonal skills
- ability to communicate complex ideas
- good presenter
- collaborative
- works well with others
- cross-functional collaboration (OK only if followed by specifics)

**Replace with**: a bullet showing what you communicated and to whom, or delete.

---

## Overused Finance/Tech Buzzwords (Use sparingly or delete)

- utilize / utilized (use "used" instead)
- interface with (use "integrated with")
- liaise / liaised (rewrite with specifics)
- facilitate / facilitated (usually vague — rewrite)
- end-to-end (OK only if followed by specifics)
- scalable (OK only if you have metrics)
- cutting-edge (delete)
- state-of-the-art (delete unless citing a paper)
- novel (OK in academic context, overused in industry)
- innovative (delete — show innovation, don't claim it)

---

## Formatting Crimes (ATS Killers)

These are not phrases but must be excluded from any resume:

- Tables for layout (columns, side-by-side sections)
- Text boxes
- Icons or emoji
- Headers/footers with key info (ATS ignores these)
- Images, logos, or photos
- Non-standard fonts (safe: Calibri, Arial, Times New Roman, Garamond)
- Colored text (except name in dark color is OK)
- Underlines on non-links
- Horizontal lines styled as table borders

---

## What to Write Instead

When you catch yourself writing a banned phrase, ask:
1. What did I actually **do**? (verb)
2. What was the **result**? (metric or outcome)
3. What **scale** was involved? (users, data size, time, money)

Example transform:
- ❌ "Responsible for data analysis and was a great team player in a fast-paced environment"
- ✅ "Analyzed 3 years of order-book tick data in Python, identifying a mean-reversion
     signal with 0.42 Sharpe ratio across 12 equity pairs"
